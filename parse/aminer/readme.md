# AMiner Network Dataset

This is the dataset from Aminer.

Descriptions can be found here https://www.aminer.cn/aminernetwork

To Download and parse the data



```
bash run.sh
```

By run the script,
We can get 5 files,
which is 

1.article_bigraph.edges : dynamic bigraph for article and authors

2.coauthor.edges : dynamic graph for coauthor relations

3.reference.edges : dynamic graph for reference relations

4.article : article id and title mapping

5.authors : authors id and name mapping
