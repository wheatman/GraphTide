# OpenStreetMap

## Data Source
https://planet.openstreetmap.org/pbf/full-history/

https://download.geofabrik.de/

A sample file ```antarctica-latest.osm.pbf``` is included under ```/pbf```


## Setup
Make sure you have installed rust or install it with:
```bash
curl https://sh.rustup.rs -sSf | sh
```

Install ```osm4routing```
```bash 
cargo install osm4routing
```

Compile ```osm4routing2pbbs```
```
make
```

## Running Code
Parse the nodes and edges information from the pbf file:
```
osm4routing pbf/antarctica-latest.osm.pbf
```
It generates ```nodes.csv``` and ```edges.csv``` in the current directory.
The graph and coordinates can be generated by running:
```
./osm4routing2pbbs
```
It generates ```graphs.adj``` and ```coords.pbbs``` in the current directory, both following the PBBS format

Graph File Formats: https://www.cs.cmu.edu/~pbbs/benchmarks/graphIO.html

Geometry File Formats: https://www.cs.cmu.edu/~pbbs/benchmarks/geometryIO.html
